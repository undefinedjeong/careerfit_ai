# PROJECT_PLAN.md — CareerFit AI 프로젝트 기획 초안



## 1. 문제 정의

취업을 위하여 어떤 스펙을 갖추어야 하는지 모르겠다.



## 2. 사용자 정의

- 대상: 학부생, 취업준비생



- 사용 시나리오:

1. 사용자가 전공, 보유 스킬, 관심 직무를 입력한다.

2. CareerFit AI가 데이터를 검색해 역량 매칭 결과를 반환한다.

3. 어떤 데이터를 기반으로 답변했는지 출처를 함께 표시한다.



## 3. 데이터 계획

- 취업 공고 컬럼: 기업명, 산업 분야(코드), 요구 역량, 요구 스킬, 요구 스택, 채용 규모, 지원 조건, 채용 일자

- 공모전 컬럼: 공모전명, 주최주관, 공공기관 관여 여부, 분야, 관련 스킬, 참여대상, 상금 규모, 참여자 수, 시행 일자

- 자격증 컬럼: 주관, 국가자격증 여부, 참여자 규모

- 데이터 출처: 강사 제공 목업 CSV (3일차)



## 4. 기능 목록

- 취업 준비 위한 활동 추천



## 5. 향후 개선 아이디어

지금은 구현하지 않지만, 나중에 해보고 싶은 것들

- 갖추어진 스펙 기반으로 현재 진행중인 추천 채용정보 출력
